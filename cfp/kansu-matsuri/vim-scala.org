# セッションの概要

Scalaを書くには

- sbt
- scala-cli
- metals

等のツールを導入し、Vim側ではLSPに関する設定を行なう必要があります。

しかし、これらのツールは時節よって変わっていきます。


本登壇ではVimでScalaを書くための環境を構築する為に使うプラグインや、それを使う方法を紹介していきます。
